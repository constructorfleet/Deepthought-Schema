| **Required**              | No               |
| **Additional properties** | Any type allowed |
| **Default**               | `true`           |
| **Defined in**            |                  |

**Description:** A named, finite set of typed values that share a common kind: every entry is either a scalar of the same dataType + unit (an enum-of-quantities) or an instance of the same model (an enum-of-records). Collections are first-class outputs of data-retrieval use cases (relation: key / relation: query) and can be referenced as the input or output of a use case the same way models are.
