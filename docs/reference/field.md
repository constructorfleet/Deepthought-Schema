## Examples

Every example below is validated against this schema as part of the docs build; if a build ships these examples, they still describe the schema correctly.

### Person Height Metric

*Source: `field/person_height_metric.yaml`*

```yaml
id: person_height_metric
name: Person Height
description: Standing height of a person measured to the nearest centimeter.
dataType: number
unit: meter
coerce: true
constraints:
  - exclusiveMinimum: 0
  - maximum: 3
  - precision:
      value: 1
      unit: centimeter
```

id: pringles_can_id
name: Pringles Can Identifier

constraints:

  - immutable: true
